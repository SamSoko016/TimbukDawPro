# Timbuk Analog Modelling Synthesizer

Created by Sam Soko

## Installation

### Windows
1. Double-click `install.bat`
2. Wait for dependencies to install
3. Run `npm start` to launch the application

### macOS
1. Open Terminal and navigate to this directory
2. Run `./install.sh`
3. Run `npm start` to launch the application

## Features

- Dual oscillators with multiple waveforms
- Voltage-controlled filter with resonance
- ADSR envelopes for amplitude and filter
- LFO with multiple modulation targets
- Built-in effects (reverb, delay)
- Real-time audio visualization
- Virtual piano keyboard
- Patch management system

## Quick Start

1. Launch the application with `npm start`
2. Use the virtual keyboard or computer keys to play notes
3. Adjust parameters using the on-screen controls
4. Save your custom patches for later use

## Keyboard Shortcuts

- `Z-M` - Play notes (Octave 3)
- `Q-P` - Play notes (Octave 4)
- `Space` - Sustain pedal
- `Ctrl+N` - New patch
- `Ctrl+S` - Save patch
- `Ctrl+O` - Load patch
- `Esc` - Stop all sounds

## System Requirements

- Windows 10+ or macOS 10.13+
- Node.js 16+ installed
- 4GB RAM minimum
- Audio output device
